# Student Club Membership Hub

## Project Title

Student Club Membership Hub

## Project Description

Student Club Membership Hub is a decentralized smart contract built on Soroban and the Stellar blockchain to manage student club memberships securely and transparently. The contract enables club administrators to add or remove members while allowing anyone to verify membership status on-chain. All membership records are stored in a decentralized and immutable manner, ensuring trust, transparency, and accountability.

## Project Vision

The vision of Student Club Membership Hub is to provide educational institutions and student organizations with a decentralized membership management system that eliminates reliance on centralized databases. By leveraging blockchain technology, clubs can maintain transparent membership records while ensuring secure administration and easy verification of member status.

## Key Features

### Membership Management

* Add new members to the club.
* Remove members when necessary.
* Maintain an accurate count of active members.

### Administrator Control

* Assign an administrator during contract initialization.
* Restrict membership management functions to the administrator only.

### Membership Verification

* Verify whether a specific address is a registered club member.
* Publicly accessible membership status checks.

### Transparency

* All membership records are stored on-chain.
* Membership updates can be audited and verified by anyone.

### Secure Access Control

* Authentication required for administrative actions.
* Prevent unauthorized membership modifications.

## Usage Instructions

### 1. Initialize Contract

Deploy the contract and assign an administrator address.

### 2. Add Member

The administrator can register new members using their Stellar wallet addresses.

### 3. Remove Member

The administrator can remove members from the club when necessary.

### 4. Check Membership

Anyone can verify whether a specific address is currently a member.

### 5. View Statistics

Retrieve the total number of active members in the club.
